# Notemp 🌠🗒️
*Private and secure, terminal-based, open-source temporary note-taking software.*

**How it works:**
- 📔 You **write** your notes!
- 🔓 You **set up an unique password**!
- 🔐 You **lock the notes**!
- 📑 You **unlock** and **read them**!
- ✈️ And when you are done, **you close the window**!
